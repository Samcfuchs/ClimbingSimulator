Infinite Recharge Climbing Simulator

Sam Fuchs (236 Alum)

This tool accurately simulates the geometry and physics of hanging multiple robots in Infinite Recharge! Set a robot's weight to 0 to remove it from the simulation. Try to keep the angle below 8 degrees! Note that the viz constrains the switch to 14.5 degrees of tilt, but still calculates the equilibrium angle if that constraint were not in place. For a robot rigidly attached to the rung, add a Center of Gravity (CG) offset. For a robot that pivots at the rung attachment, set the CG offset to 0. This project is hosted on Github.
